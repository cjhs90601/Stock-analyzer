記帳通設定

首次使用需要設定 Google OAuth Client ID

快速設定步驟：

  1. 前往 Google Cloud Console，建立或選取一個專案
  2. 啟用 Gmail API 與 Google Sheets API
  3. 建立 OAuth 2.0 憑證（類型：Web 應用程式）
  4. 授權 JavaScript 來源加入：http://localhost:8080
  5. 複製 Client ID 貼到上方欄位
  6. 用以下指令啟動伺服器後開啟 http://localhost:8080：
    python -m http.server 8080
